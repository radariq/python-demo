Path Tracing Demonstration
============================

This sample application is a simple example of tracing
a path using the RadarIQ sensor.

![Screenshot](demo.png).

Installation:
-------------
1. Create a virtual env for this project

``python -m venv venv``

2. Activate the virtual env

* Windows: ``venv\scripts\activate``
* Linux: ``source venv/bin/activate``

3. Install requirements
``pip install -r requirements.txt``

OR:

Simply click on the "install" batch file, and it will install
all the files required.

Usage:
---------

1. Run the application
      enter ``python tracing.py`` in cmd after installation

2. Walk around and see the path you took traced out.
